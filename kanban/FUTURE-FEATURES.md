# Kanban Board — MVP Sonrası Özellikler

Bu dosya, MVP kapsamı dışında bırakılan fikirleri saklar.
MVP tamamlandıktan sonra bu listeye dönebiliriz.

---

## Görev Zenginleştirme
- Görev içinde alt görevler (checklist)
- Deadline / bitiş tarihi alanı
- Etiket/tag sistemi (birden fazla label)
- Görev detay modalı (açıklama, ek notlar, geçmiş)
- Markdown desteği görev açıklamalarında

## İş Akışı
- Sütun içinde sıra değiştirme (drag pozisyonu)
- Filtrele: önceliğe, ilerlemeye, tarihe göre
- Arama: görev metni içinde arama
- Toplu seçim ve toplu taşıma

## Claude Code Entegrasyonu
- Claude'a komut gönderme butonu (/start, /sync, /wrap-up)
- Oturum durumu göstergesi (hangi komut çalışıyor)
- memory.md tam görünümü (collapsible panel)
- knowledge-base.md quick view

## Görselleştirme
- Dark mode toggle
- Sprint burndown chart
- Kolon bazlı istatistikler (tamamlanma oranı, ortalama süre)
- Zaman çizelgesi / Gantt görünümü
- Dashboard: toplam task, haftalık tamamlanan, blocker sayısı

## Ayarlar & Konfigürasyon
- UI üzerinden kolon yönetimi (ekle, sil, yeniden sırala)
- Kolon renk seçici
- Proje adı ve açıklama düzenleme

## İşbirliği (Multi-user)
- Göreve atanan kişi
- Yorum sistemi
- Bildirimler
- Paylaşılabilir link

## Dışa Aktarma
- CSV / JSON export
- Markdown özet raporu
- PDF çıktı

## Entegrasyonlar
- GitHub Issues senkronizasyonu
- Linear / Jira import
- Slack bildirim webhook'u
